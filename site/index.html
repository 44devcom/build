--- layout: home title: Hacker Theme Demo --- This repository demonstrates the official GitHub Pages `pages-themes/hacker` remote theme. Features included in this demo: - Responsive header and nav - Author / social area - Posts list and pagination (sample) - Feed link via `jekyll-feed` Run locally: ```bash bundle exec jekyll build --config _config.yml --source site --destination _site --future python3 -m http.server 8080 --directory _site ```
