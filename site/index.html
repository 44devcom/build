Usage: Expose with Cloudflared

Theme feature parity

To approximate the GitHub Pages Hacker theme, this demo includes:

  • Green-on-black color palette and monospace typography
  • Simple responsive layout (header, content, footer)
  • Feed link via `jekyll-feed` (declared in `_config.yml`)

Run locally

  1. Start a local server from the `site` folder on port 8080:
    2. cd site
./serve.sh
  2. Expose it with the tunnel script (Termux/cloudflared):
    3. bash t.sh
